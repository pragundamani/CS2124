# C++ Learning Path - Complete Notes

A comprehensive collection of C++ notes organized from beginner to expert level, covering fundamental concepts to advanced topics.

## üìö Learning Path

### 1. **basics.md** (752 lines)
**Start here if you're new to C++**

Topics covered:
- Program structure, compilation process, and syntax
- Preprocessor directives and namespaces
- Data types, variables, and constants
- Operators (arithmetic, comparison, logical, bitwise)
- Control flow (if/else, switch, ternary)
- Loops (while, for, do-while, range-based)
- Input/Output and stream manipulators

### 2. **advanced_basics.md** (1,507 lines)
**Continue here after mastering basics**

Topics covered:
- Functions (declarations, prototypes, overloading, default parameters)
- Recursion (factorial, fibonacci, binary search, tail recursion)
- Arrays (static, multidimensional, passing to functions)
- Vectors (std::vector operations, 2D vectors, capacity vs size)
- Pointers (basics, null pointers, pointer arithmetic)
- Structs (definition, initialization, vectors of structs)
- Enumerations (enum and enum class)
- Standard Library Algorithms (sort, find, count, accumulate, reverse)
- Best practices (magic numbers, naming, error handling, DRY principle)

### 3. **common_patterns.md** (1,216 lines)
**Practical patterns and real-world examples**

Topics covered:
- String operations (comparison, substr, find, conversion)
- Vector patterns (initialization, 2D vectors, operations)
- Modern loop patterns (range-based, iterator-based, algorithm-based)
- File I/O (reading, writing, binary mode, error handling)
- Common coding patterns (two-pointer, sliding window, frequency counting)
- Advanced patterns (finding duplicates, custom sorting)

### 4. **intermediate.md** (1,479 lines)
**Deep dive into memory and OOP**

Topics covered:
- Pointers & Memory (declaration, dereferencing, pointer arithmetic)
- Dynamic Memory Management (new/delete, memory leaks, RAII)
- Smart Pointers (unique_ptr, shared_ptr, weak_ptr)
- Object-Oriented Programming (classes, constructors, destructors)
- Access specifiers (public, private, protected)
- Static members and const member functions
- Standard Library Containers (array, vector, deque, list, forward_list)
- Associative Containers (map, set, unordered_map, unordered_set)
- Exception Handling (try-catch, custom exceptions, exception safety)
- Templates Basics (function templates, class templates, specialization)

### 5. **advanced.md** (1,607 lines)
**Master advanced C++ features**

Topics covered:
- Advanced OOP (inheritance, polymorphism, virtual functions, abstract classes)
- Operator Overloading (arithmetic, comparison, stream, subscript operators)
- Move Semantics & Rvalue References (lvalues vs rvalues, move constructors)
- std::move and perfect forwarding
- Advanced Templates (specialization, variadic templates, SFINAE, concepts)
- Lambda Expressions (capture modes, mutable lambdas, generic lambdas)
- STL Algorithms Advanced (transform, partition, unique, set operations, heap)
- Iterators (categories, operations, custom iterators)
- Multithreading (std::thread, mutex, condition variables, atomic operations)

### 6. **expert.md** (1,651 lines)
**Cutting-edge C++ and professional practices**

Topics covered:
- Modern C++ Features (C++11/14/17/20)
  - auto, decltype, constexpr, nullptr
  - Structured bindings, std::optional, std::variant, std::any
  - Ranges library, coroutines, modules
- Advanced Memory Management (custom allocators, memory pools, placement new)
- Metaprogramming (template metaprogramming, compile-time computation)
- SFINAE techniques and policy-based design
- Design Patterns (Factory, Singleton, Builder, Adapter, Decorator, Observer, Strategy)
- SOLID Principles
- Performance Optimization (profiling, cache optimization, SIMD, compiler flags)
- Build Systems (Makefiles, CMake)
- Professional Tools (Git, static analyzers, debuggers, sanitizers)
- Testing (Google Test, Catch2, TDD, code coverage)
- Low-Level Programming (bitwise operations, endianness, inline assembly)
